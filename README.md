# Lakme Academy Website

A modern, responsive website for Lakme Academy - a professional beauty and cosmetology institute offering comprehensive training in hairstyling, cosmetology, makeup artistry, and skincare.

## 🌟 Features

- **Responsive Design**: Mobile-first approach with complete mobile, tablet, and desktop optimization
- **Modern UI**: Clean and professional interface with smooth animations
- **Course Pages**: Detailed information for each course (Hairstyling, Cosmetology, Makeup, Skincare)
- **Contact Form**: Integrated contact form with validation
- **Accessibility**: ARIA labels and semantic HTML5 for better accessibility
- **Navigation**: Sticky header with smooth scrolling and active link highlighting
- **Performance**: Optimized for fast loading and smooth user experience

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side dependencies required

### Installation

1. **Clone or Download** the project files
```bash
git clone <repository-url>
cd lakme
```

2. **Open in Browser**
   - Simply open `index.html` in your preferred web browser
   - Or use VS Code's Live Server extension for development

### Using Live Server (VS Code)

1. Install the Live Server extension in VS Code
2. Right-click on `index.html` and select "Open with Live Server"
3. Your browser will open with the site running on `http://localhost:5500`

## 📝 Customization

### Updating Content
1. Edit HTML files directly to update text content
2. Update course information in respective course pages
3. Modify contact details in the contact section

### Changing Colors
1. Update CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #8B0000;
    --secondary-color: #2c3e50;
    --accent-color: #f39c12;
    /* ... more variables */
}
```

### Adding Images
1. Place images in the `images/` folder
2. Update the course cards with background images
3. Ensure images are optimized for web
